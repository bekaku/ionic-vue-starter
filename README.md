# Ionic vue Starter


Starter template for Ionic + Vue js3 + TypeScript project
## Plugins
1. vue-i18n 
2. vue-router
3. vuex
4. js-cookie

## Getting Started

1. Clone this repo
2. `npm install`
3. `ionic serve` or `npm run serve` or `yarn serve`
4. `ionic build` or `npm run build` or `yarn build`
